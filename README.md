# Task 1 - Linear Regression Model

## 📌 Project Overview
This project builds a Linear Regression model using the California Housing dataset.

## 🧠 Steps Performed
- Data Loading
- Exploratory Data Analysis (EDA)
- Train-Test Split
- Model Training
- Model Evaluation (MAE, RMSE, R²)
- Visualization

## 📊 Results
- R² Score: ~0.60
- Model performs reasonably well on housing data.

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
